base アドレスの差分 変更値  
  
[]ごとに項目が追加されます。  
複数個処理がある場合はその下に追加していってください。  
  
例  
[即クリア]  
base 0x100023c 0  
base 0x1000240 0  
[無敵]  
base 0x10543c4 100  
  
今後追加予定  
  
アドレスを取得してパッチする  
search サーチ範囲-サーチ範囲 数値  
get(x) アドレスの差分 数値  
  
複数回サーチしてからアドレスを取得する  
search サーチ範囲-サーチ範囲 数値 & search 数値 & search 数値  
get(x) アドレスの差分 数値  
  
&で区切ればその回数分サーチする  
サーチ範囲に指定がなければ0x1-0x200000000の範囲に自動設定  
複数個の場合サーチ範囲は最初の1回のみ  
xには複数個ヒットがあった場合に上から何番目のアドレスを取得するか
