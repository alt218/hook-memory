# txtメモリの使い方
[名前]  
base アドレスの差分 変更値  
[名前]  
base アドレスの差分 変更値  
base アドレスの差分 変更値  
  
[]ごとに項目が追加されます。  
複数個処理がある場合はその下に追加していってください。  
  
例  
[即クリア]  
base 0x100023c 0  
base 0x1000240 0  
[無敵]  
base 0x10543c4 100  

`base` 形式で `val` は使用するユーザが自分で数値を打ち込む
base 0x4 val  

  
コマンド検索でアドレス取得してパッチする  
  
search サーチ範囲-サーチ範囲数値 & 検索数値 & 検索数値  
get(x) アドレス差分数値  
  
例1  
search 0x100000000-0x200000000 5  
get(1) 0x00 9  
  
例2（複数回検索）  
search 0x100000000-0x200000000 100 & 90 & 80  
get(2) 0x04 777  

例3（手入力値を使う）  
search 0x100000000-0x200000000 100  
get(1) 0x04 val  
  
ルール  
- `&` で区切った回数だけ検索を続行  
- 範囲指定が無い場合は `0x1-0x200000000` を自動使用  
- 複数検索時に範囲を指定できるのは最初の1回のみ  
- `x` はヒット一覧の上から何番目を使うか（1始まり）  
- `base` 行は `userInfo[@"base"]` を基準に `base + 差分` へ書込み
